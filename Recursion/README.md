# 🔄 Recursion in Python (Mastery Guide)

---

## 📚 What is Recursion?

Recursion is a programming technique where a function calls itself to solve smaller instances of the same problem. It continues until it hits a **Base Case** (the stopping condition), preventing infinite loops and stack overflow errors.

> **Analogy included in the notebook:** The Unix directory structure and the beautiful self-similarity found in geometric Fractals!

## 🛠️ Code Challenges Covered

The notebook contains practical, hands-on implementations for the following problems:

### 1. Basic Math & String Operations
* **Factorial Calculation:** Finding $n!$ recursively.
* **String Reversal:** Reversing a string utilizing the call stack memory.
* **Fibonacci Series:** Generating numbers using multiple-recursion ($F(n) = F(n-1) + F(n-2)$).
* **Sum of Natural Numbers:** Calculating the sum of all natural numbers up to $n$.
* **Power Calculator:** Calculating $Base^{Exponent}$ mathematically.

### 2. Array & String Manipulation
* **Frequency Counter:** Counting the absolute frequency of a target value in an array.
* **Palindrome Checker:** A deeply nested boolean check to see if a string reads identically forwards and backwards.
* **In-Place Array Reversal:** Reversing an array without using external arrays or loops.
* **String Permutations:** An advanced multiple-recursion algorithm to find all permutations of a string.

### 3. Math & Logic Puzzles
* **Sum of Digits:** Extracting and summing digits of a number recursively.
* **Perfect Square & Cube Verification:** Using recursive binary search patterns to verify squares and cubes.

### 4. Advanced Structural Algorithms
* **Flatten a Nested Dictionary:** Deep-traversing a nested string-to-object dictionary into a single-level layout using dot notation (`parent.child`).
* **Balanced Parentheses Generation:** An execution tree algorithm to generate all valid combinations of $n$ pairs of brackets.
* **The Tower of Hanoi:** A multiple-recursive breakdown that solves the classic puzzle in the minimal number of steps ($2^n - 1$).
